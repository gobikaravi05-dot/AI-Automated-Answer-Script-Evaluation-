# Automated Answer Script Evaluation

## 📌 Project Overview

Automated Answer Script Evaluation is an AI-powered web application that evaluates handwritten student answer sheets automatically. The system uses OCR to extract text from scanned answer sheets and AI to evaluate answers based on the question paper and optional answer key. It generates question-wise marks, total marks, and provides downloadable transcripts.

---

## ✨ Features

* Faculty Login
* Student Login
* Upload Question Paper
* Upload Student Answer Sheets
* Upload Answer Key (Optional)
* OCR-based Handwritten Text Extraction
* AI-powered Answer Evaluation
* Question-wise Marks Calculation
* Total Marks Generation
* Faculty Dashboard
* Student Dashboard
* Download Transcript
* Download Evaluation Results
